# Previsão do Consumo de Água em Data Centers com Machine Learning

Este projeto foi desenvolvido para a disciplina de Inteligência Artificial e utiliza técnicas de Machine Learning para prever o consumo diário de água em data centers a partir de informações operacionais, comparando o desempenho dos algoritmos **Regressão Linear** e **Random Forest Regressor**.

## Objetivo

Avaliar a capacidade dos modelos de prever o consumo diário de água e identificar as variáveis que mais influenciam essa previsão.

## Dataset

- **Global Data Center & AI Water/Electricity Usage (2019–2025)**
- Fonte: https://www.kaggle.com/datasets/ashyou09/global-data-center-and-ai-waterelectricity-usage

## Tecnologias utilizadas

- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Resultados

Os resultados indicaram que o **Random Forest** apresentou desempenho superior ao da **Regressão Linear**, obtendo menores erros de previsão e maior coeficiente de determinação (R²), embora com maior custo computacional.
